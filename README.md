# cast

## Dependency
- server: sails v0.12.13
- node: v7.6.0
- database: mysql
- session: redis
- frontend: react v15.6.1

## Javascript Style Guide

Make sure your IDE respect the settings in `.editorconfig` and use the linting guideline in `.eslintrc.json`.
